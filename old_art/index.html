about

Some old (old) images, but what can't be improved by a fancy website?

These image are built using:

  • Context Free - generate (2D) images from a stochastic grammar
  • Structure Synth - essentially a 3D version of the above. See also Mikael Hvidtfeldt Christensen's fantastic blog Syntopia
  • Sunflow - Old, but fun, JAVA based raytracer
  • LuxRender - Physically based renderer.
